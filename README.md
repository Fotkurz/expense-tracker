# Expense Tracker API

Simple backend app written in Kotlin + Spring boot
for tracking expenses for the roadmap.sh project:

- https://roadmap.sh/projects/expense-tracker-api

Feel free to copy the code or use it in any way you need, just
make sure to recommend the original repo if it's useful for you
somehow.

## Specs

- Kotlin 2.2.21 (Java 24)
- Gradle Kotlin for build
- Spring Boot 4.0.1
- PostgreSQL (Spring Data JPA + Hibernate)
- Hexagonal Arch (kinda)

Used https://start.spring.io/ for initial setup.
